# 🛍️ TrovéMart - Online Shopping Platform (Server)

Welcome to the TrovéMart Backend Server.

This is the backend repository for the TrovéMart Online Shopping, where buyers and sellers connect to trade products securely and seamlessly. It handles user authentication, product moderation, wishlist management, reviews, and order logic.

---

## 🌐 **Client Side Repository** 
## Frontend/Client: https://github.com/ORNOB-083/TroveMart-Online-Shopping-Platform 

---

## 🚀 Key Features

-   **Secure Authentication**: Local login/register with JWT tokens, plus Google/Facebook OAuth integration.
-   **Product Moderation**: Admin approval workflow ensures only high-quality/approved items are shown to the public.
-   **Wishlist & Favorites**: Users can save products and instantly see their saved items.
-   **User Reviews & Ratings**: Built-in rating system that dynamically updates a product's average score based on user feedback.
-   **Admin Panel Backend**: Endpoints for banning users, reviewing pending items, and managing the platform ecosystem.
-   **Seller Support**: Database structures ready for users to onboard as sellers.

---

## 🛠️ Tech Stack

-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Runtime**: [Node.js](https://nodejs.org/)
-   **Framework**: [Express.js](https://expressjs.com/)
-   **Database**: [MongoDB](https://www.mongodb.com/) (Native Driver)
-   **Auth**: JSON Web Tokens ([JWT](https://jwt.io/))
